使用说明书终极形态 · v∞

假设你拿到的是 100% 完成的太一 Ω。这是它的说明书。

一、它是什么

太一 Ω 是一个认知操作系统：库（pip install theone）、服务（自托管节点）、标准（认知凭证协议）三种形态同体。它不取代你正在用的任何大模型——它把它们变成自己的感官，同时把因果推理、终身记忆、校准元认知与可验真凭证赋予它们。

二、安装与进入

  1. 普通用户：浏览器打开任意太一节点 → 地址栏「安装应用」即可作为 PWA 安装到手机/电脑（本演示就是这样安装的）。
  2. 开发者：pip install theone → 三行接入：one = TheOne(base="any-open-llm") → one.ask(...)，返回 answer + credential。
  3. 企业：helm install theone 私有化部署，数据不出域，审计仪表盘全量检索凭证。
  4. 科研：theone bench 一键复跑全部预注册实验；仿真农场即服务。

三、核心操作

  • 问因果：直接问「X 会导致 Y 吗」。它会区分关联与干预，给出 P(Y|do(X)) 与因果图。
  • 问反事实：「如果当时…会怎样」。三步法（溯因→干预→预测）推演，给出方向与置信。
  • 验凭证：每个重要回答下方点「查看凭证」→ 因果路径 / 调整集 / 记忆引用 / 七轴置信 / 图哈希。用独立校验器 theone verify cred.json 一行验真——校验器与生成器零共享代码。
  • 管记忆：「记住…」即写入（带来源与时间戳）；记忆面板里查 / 改 / 删 / 一键带走。
  • 接受拒答：当它说「这个问题在我的认知边界外（不可说）」——这不是故障，这是它最贵的功能。
  • 开实验：对它说「去验证这个假设」，它会预注册判据、设计最小干预、在沙箱执行并把结果（无论正负）原样给你。

四、配置火候（文化先验）

在设置中选择价值先验：中立 / 儒家 / 道家 / 64卦离散空间 / 自定义。先验透明可审计——它影响表达的「火候」，不黑箱地篡改事实。

五、能力清单（终极形态）

  • 语言/代码/多模态：复用最强开源基座（热插拔，换基座灵魂不动）
  • 因果：原生 do 算子、反事实、自动实验验证
  • 记忆：跨年终身记忆，可学习记忆核，用户主权四权
  • 元认知：七轴不确定性 + 七值真值 + 强制校准，敢说不知道
  • 凭证：每个重要输出机器可验真（认知界的 HTTPS）
