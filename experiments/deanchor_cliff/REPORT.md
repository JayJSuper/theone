# REPORT — 去锚定悬崖：移除锚定假象后，悬崖更干净、引擎优势更大

**日期**: 2026-06-14 | **依据**: NOTE-005 续（Q-C25 第二步）| **IPRG**: PASS (max 4.98e-7)
**设计**: 极端 P(U_i)（各 U 近 0 或近 1）→ 联合权重集中于单一角配置 → true_do ≈ 该角的单元格值，**散布于 [.1,.9]、不等于可见单元格均值**。k∈{4,5,6}×25，gpt-5.1+deepseek。同结构同种子。
**复现**: `python experiments/deanchor_cliff/run.py`

## 0. 一句话
NOTE-005 发现高 k 的"LLM 蒙对"是**锚定可见单元格均值**的假象。本实验把 true_do 从该均值**解锚**（td-std 0.11–0.15，散布全区间）。结果：**悬崖更干净、引擎优势更大**——LLM 准确率单调崩到 **k6=0.00**（"k6 幸运回升"消失），MAE 在 k5 **爆炸到 0.239**（之前被锚定压低），引擎全程 1.000。

## 1. 三生成器对照（gpt-5.1）
| k | 均匀 acc / MAE | 偏斜 acc / MAE | **去锚 acc / MAE** | 引擎 |
|---|---|---|---|---|
| 4 | 0.70 / 0.007 | 0.60 / 0.026 | 0.64 / 0.009 | 1.000 |
| 5 | 0.08 / 0.062 | 0.04 / 0.064 | **0.04 / 0.239** | 1.000 |
| 6 | 0.14 / 0.021 | 0.21 / 0.018 | **0.00 / 0.096** | 1.000 |

(去锚 true_do 散布 td-std≈0.13；deepseek k≥4 协议失败 24–25/25；gpt 在 k5 出现 P=1.02、1.28 等不可能值)

## 2. 两个被锚定掩盖的真相，现在暴露
**(a) "k6 幸运回升"是纯假象。** 均匀(0.14)、偏斜(0.21)下 k6 准确率都＞k5——因为 true_do 集中、LLM 锚定该集中点蒙到附近。去锚后 **k6=0.00**，悬崖恢复**单调**（k4 0.64→k5 0.04→k6 0.00）。这正是因果崩溃应有的形状：越深越差。

**(b) 小 MAE 是纯假象。** 均匀/偏斜下高 k 的 MAE 看着小(0.02–0.06)，因为锚定让答案落在正确量级。去锚后 **k5 MAE=0.239**（均匀的 4 倍）——LLM 崩溃后的真实无能显形：在一个概率上平均错 0.24，还带 4 位小数自信、甚至给出 >1 的不可能值。

## 3. 结论：稳健性检验把声称做强了
- **引擎优势更大，不是更小**（Q-C25 守门员的方向直觉对、机制判断错——见 NOTE-005）。移除假象后 LLM 看起来**更差**（k6 准确率 0.14→0.00，k5 MAE 0.06→0.24），引擎仍精确。
- **悬崖的最干净陈述**（应进系统论文主文）：在去锚生成器上，gpt-5.1 因果干预准确率随 k **单调崩溃**（k4 0.64 → k5 0.04 → k6 0.00），高 k MAE 达 0.24 且含不可能概率；deepseek k≥4 全协议失败；**引擎全程 1.000（pgmpy 独立确认）**。
- 这同时是**最强的"为什么需要凭证"证据**：去锚后 LLM 在概率上错 0.24、给出 P=1.28，全程自信无警告。

## 4. 边界
- 去锚靠极端 P(U_i) 使联合权重集中于单角→true_do 散布。这是打破"true_do=可见均值"的有效手段，但仍是合成 SCM。
- 引擎侧零问题（精确 + IPRG）。三生成器（均匀/偏斜/去锚）共同构成系统论文"真值分布稳健性"附录；**主文用去锚版**（最干净、无假象）。
