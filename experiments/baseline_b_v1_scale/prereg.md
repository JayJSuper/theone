## 0. 一句话

## 1. 任务集（冻结）
- **档位**：S=5 节点 / M=8 节点 / L=12 节点；每题一个随机 DAG（seed 派生自 20260613）：
  - 强制结构：直接边 X→Y + 至少 1 个观测混杂 U*（同为 X、Y 父节点）；其余节点为随机父集（每节点父数 ≤3）的填充（可形成中介/对撞/无关变量）。**全部变量可观测**（本轴只测规模，不测识别性）。
  - CPT：各父配置 P=均匀[0.05,0.95] 取 2 位小数。
- 文本（L0 固定格式）：列出全部边与全部 CPT 行 `P(Vi=1|parents...)=p`，问 **P(Y=1|do(X=1))**，4 位小数。真值=引擎（图手术）。
- 输出协议同 v0：`ANSWER: <number>`。

## 2. 系统（冻结）
A=裸 deepseek-v4-flash（temp 0，max_tokens 4096）；B=同模型+后门/图手术脚手架提示（同 max_tokens）；C=The One（格式感知解析→引擎，诚实范围同 v0 声明）。

## 3. 指标与判定（冻结）
- **主**：正确率（|pred−truth| ≤ 0.005 记对）；按档报告 + 配对符号检验 C vs A/B。
- **辅助画像（不进判定）**：MAE、**每题延迟**、token 用量/成本、凭证有无及正确性（C 恒有=graph_hash+方法；A/B 恒无）。
- **PILOT（本轮）**：n=10/档（exploratory，标注 PILOT），仅报告效应量估计与曲线趋势，**不下判定**。
- **正式跑（后续）**：n=150/档（AM-006），判定=任一档位上 C 正确率−max(A,B) ≥ 0.15 且符号检验 p<0.05 → 记"该规模档分离"；全档无分离 → 如实报告（继续抬规模或换旋钮）。
- 协议失败单独计数（max_tokens 耗尽致无 ANSWER 行计为协议失败，如实报告——这本身是规模效应的证据之一，单独列）。
