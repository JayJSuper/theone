# REPORT — 半结构化野生文本：赌注①核心主张的公平隔离证明

**日期**: 2026-06-13 | **冻结**: AM-014（反常识 option A）、AM-015（本设计） | **真实度**: 【已验证】
**前情**: WEAK-03 指出"无数字叙事"的数值对照 ill-posed（引擎无 CPT 可算）。本实验把数字**埋进 prose**（现实因果问题本就带数据），使对照良定，并做到**最干净的单变量隔离**。

## 1. 设计：同一前端，只差"谁做边缘化"
- 24 条自洽叙事（12 常识 / 12 反常识），每条把一个 3 节点混杂 SCM 的 7 个条件概率以**自然语言百分比**写进 prose；并埋入混杂陷阱：保证 |do − obs| ≥ 0.06（实测均值 gap=0.094）。`P(X|U)` 作为**干扰项**写入（对 do(X=1) 无关）。
- **两臂用同一个模型 gpt-5.1**：
  - **RAW**：读叙事 → 直接给 P(Y=1|do(X=1))（脑内抽取+边缘化）。
  - **The One**：gpt-5.1 把{结构+7 个数字}抽成 JSON → 确定性 `InterventionEngine` 做精确后门边缘化。
- 前端完全相同 ⇒ **任何差距 100% 归因于"把组合边缘化卸载给精确引擎"**。这正是赌注①核心主张的裸隔离。

## 2. 结果（n=24，可复现 `results.json`）
| 指标 | RAW gpt-5.1（脑内） | The One（引擎边缘化） |
|---|---|---|
| 严格命中 ±0.005 | **1/24 (0.042)** | **24/24 (1.000)** |
| 宽松命中 ±0.05 | 11/24 (0.458) | 24/24 (1.000) |
| MAE | **0.0649** | **0.000000** |
| 最大误差 | **0.211** | 0.000000 |
| 抽取有效率 | — | **100%** |

分层：常识 raw_MAE=0.068，反常识 raw_MAE=0.062（方向无关）。

## 3. 这是什么、不是什么（诚实定性）
- **不是"LLM 读不懂数字"**：抽取有效率 100%——gpt-5.1 完全能把 prose 里的百分比读出来。
- **不是"LLM 天真混杂"**：陷阱指标 = 0/24——gpt-5.1 **知道**要做干预校正，从不偷懒报观测值。
- **也不是系统性欠校正**：只有 4/24 落在 truth 与 obs 之间；最差的 item 21 raw=0.512 反而**冲过了** truth(0.723)。
- **真相是**：误差是**双向散布的算术漂移**——gpt-5.1 知道该怎么做、数字也读对了，但**在脑内把"按 P(U) 加权求和"这步算糊了**，MAE 0.065、最大 0.211。把这一步交给引擎，立刻精确。

> **赌注①核心主张，隔离证明成立**：认知层的价值 = 把因果计算（哪怕只是 k=1 的两项加权）从推理链移出、交给确定性引擎。差异不在知识、不在抽取，**纯在计算的可靠性**。

## 4. 与组合爆炸悬崖的连接
本实验是 **k=1**（最易的边缘化，仅 2 个配置）。即便如此，脑内算已漂移 MAE 0.065。组合爆炸悬崖实验已证 k≥5 时脑内算**崩溃**（准确率→0）。两者是**同一条曲线**：本实验是它的"低 k 脚趾"，悬崖是它的"断崖"。预期 The One 的领先随 k 单调放大（见 §6 后续）。

## 5. 诚实边界（随结论同行）
1. **数字陈述是干净的**（标注清晰的条件百分比）→ 抽取 100% 是这个前提的产物。更杂乱的真实 prose 会压力测试抽取本身（那是 LLM 的活、是另一种失效模式，本实验未覆盖）。
2. **k=1 单混杂**：本实验证的是"即使最易场景脑内算也不可靠"，领先随 k 放大需 §6 验证。
3. **The One 的 1.000 是真的引擎计算**（非 WEAK-03 那种硬编码）：引擎在 gpt-5.1 **自己抽出的数字**上独立运行；100% 精确仅因本批抽取全部正确——抽取错则 The One 也错（AM-007 协议失败计为错），这是真实风险面。

## 6. 后续
- **k=2 / k=3 扩展**：数字埋更多混杂，验证 The One 领先随组合数放大、raw 滑向悬崖。
- **杂乱 prose 抽取压力测试**：把数字写得更含糊，单独测抽取失效率（LLM 前端的真实瓶颈）。
- 守门员 Jack 复核本设计的隔离严谨性（同前端是否真排除了所有混淆）。

**一句话**：把因果计算交给引擎，不是因为 LLM 笨，而是因为**算术的可靠性不该赌在自回归采样上**——这是 The One 在"数字已给定"主场最锋利、最诚实的一刀。
