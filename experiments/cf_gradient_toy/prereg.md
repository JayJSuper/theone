## 0. 一句话
检验「在双头 MLP 上增加反事实损失项（λ>0）」是否比「纯事实训练（λ=0）」更能学到**干预（do）效应**——作为反事实梯度线的玩具地面演习，为 T4 跨族迁移判决门铺垫。**本玩具不证明 T4，只观测体征 1 级。**

## 1. 目标（冻结）
- 主问题：λ>0 的反事实监督是否让模型在**留出干预任务**上的预测误差显著低于纯关联基线。
- 体征观测：训练是否「有动静」（体征 1 级），干预误差是否优于基线 BCa 下限（体征 2 级）。

## 2. 数据生成（冻结）
- SCM 族：线性高斯三节点（复用 `SyntheticSCM`，U,X,Y）。
- 参数抽样（每实例独立 seed）：p=β_ux·β_uy ∈ [0,0.8] 对称分解 β_ux=β_uy=√p；β_xy ∈ [0,0.5]；noise ∈ [0.1,0.5]，均匀抽样。
- 每实例：观测样本 n=500，计算特征向量
  φ = [Cov(X,Y), Var(X), Cov(X,U), Cov(U,Y), Var(U)]（5 维，标准化）。
- 标签：
  - 事实标签 y_fac = 观测斜率 Cov(X,Y)/Var(X)（头1 目标）。
  - 反事实标签 y_cf = 真 do-ATE = β_xy（头2 目标，由合成 SCM 可控噪声提供）。
- 训练族 vs 留出族：**不相交 seed 区间**（训练 base_seed=300000，留出 base_seed=700000）。

## 3. 模型与训练（冻结）
- 双头 MLP：trunk 5→16→16（tanh），头1/头2 各 16→1 线性；numpy 手写前向+反向，无第三方深度学习依赖。
- 损失 L = MSE(head1, y_fac) + λ·MSE(head2, y_cf)。
- 条件：**对照 λ=0**；实验 λ ∈ {0.5, 1, 2}。
- 每条件 **20 seeds**（控制权重初始化 + 数据抽样）。
- 优化：全批量 Adam，lr=0.01，epochs=300，He 初始化；一切 seed 显式。
- 训练集实例数=512，留出集实例数=256。
- **预算上限**：单次全程 CPU ≤30 分钟；超时即中止并报告。

## 4. 判据（冻结，不可为出体征而改）
- **体征 1 级（T4-S12）**：某条件下，存在 seed 使训练中**连续 3 个 epoch 两个头损失各相对下降 ≥1%** → 记"体征①出现"（报告出现该体征的 seed 比例）；全条件无 → 记"本轮无感，继续站"（**合格汇报**）。
- **体征 2 级**：实验条件（λ>0）的留出干预 MSE < 纯关联基线（预测 do-ATE=观测斜率）的 bootstrap BCa 95% CI **下限**（B=1000）→ 单独标注。
- 对照预期：λ=0 头2 无监督，预期**不**出现头2 体征；若意外出现需存疑排查（不得美化）。
