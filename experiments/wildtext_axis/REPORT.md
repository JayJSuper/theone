## 2. v2 失败的实质构成（抽样分析，按判据要求）
1. **角色漂移（生成侧，真问题）**：部分叙事改写后，双抽取器**一致地**把处理变量认成别的——文本本身让因果角色漂了。
2. **先验知识干扰（预注册已声明的局限）**：如"维生素D/户外时间"案例——现实世界里户外时间确实导致维生素D，抽取器的先验与文本钦定的角色打架。**正式跑必须上反平衡设计**（随机化角色指派）。
3. **L2 方向迷失（真难度）**：观察性措辞下抽取器无法定向 treatment/outcome——这正是 L2 被设计出来的难度，是被试将面对的同款挑战。
4. 残余工件 ~1-2 例（个别解析失败）。

## 3. 给 Q-C17 协议的可行性结论（回守门员）
- **结构级回译机制本身可运行**（CC 修正案的物证成立）。
- 但"双 LLM 交叉校验 + 回译"在 v2 条件下只能认证 ~58% 的叙事——**85% 阈值用纯自动化达不到**。剩余 42% 按协议进人工校对队列（或：反平衡+更强抽取协议后重测）。
- **判定（按预注册）**：自动化语料认证暂停；正式野生文本轴的语料需"自动通过的 58%"+"人工校对补足"双通道，或先解决反平衡设计再重测验收率。
