## 3. 诚实边界（预注册写死，必须重申）
- 语义 regime 是**平局**不是"余弦胜"——任务太易，三核都满分。真正的分离在移位与绑定。
- **关键限制**：本实验只证明"各核有自己碾压基线的 niche"，**不证明**它们在真实记忆任务上比"余弦 + 足够数据/足够维度"更划算。后者需端到端任务，不在本轮。频谱的移位不变性、HRR 的结构绑定，是否对 The One 的实际记忆负载有用，取决于负载里是否真有"移位模式""角色-填充结构"——这是 M2 真实接入时才能回答的问题。

## 4. 采纳决定
- **spectral / hrr 升为"条件采纳候选"**：当 M2 记忆负载中出现周期/移位模式或角色-填充结构时启用；否则余弦默认（最简、够用）。
- 与 BF-01（洛书）对照：洛书在它声称的主场（2D 平衡）也输了→否证；检索核在各自主场赢了→条件采纳。**同样的基准纪律，不同的诚实结论。**
