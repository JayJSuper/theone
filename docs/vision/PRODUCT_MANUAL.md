## 1. 它是什么 / 不是什么

**是**：认知操作系统——三种形态同体：
- **库**：`pip install theone`，嵌入一切应用；
- **服务**：自托管节点 / 社区节点，数据不出门；
- **标准**：认知凭证协议（如同 HTTPS 之于网络）。

**不是**：又一个基座大模型（不与任何模型竞争参数规模）；不是 LLM 套壳（判别三问：脚手架两周能近似的不做、基座下版会内置的不做、不让凭证更可信的不做）。

## 3. 五个核心动作

1. **问因果**：「X 会导致 Y 吗？」→ 区分相关与因果，给 P(Y|do(X)) 与因果图。
2. **问反事实**：「如果当时…会怎样？」→ 溯因-干预-预测三步，附区间与置信。
3. **验凭证**：回答下方「查看凭证」→ 因果路径/调整集/记忆引用/七轴置信/图哈希；命令行 `theone verify cred.json` 一行验真（校验器与生成器零共享代码）。
4. **管记忆**：说「记住…」即写入（带来源时间戳）；记忆面板四权：**查 / 改 / 删 / 带走**。
5. **开实验**：「替我验证这个假设」→ 预注册判据 → 最小干预 → 沙箱执行 → 无论正负原样报告。

## 4. 两个需要适应的设计

- **它会拒答。**「这在我的认知边界外（不可说）」不是故障，是它最贵的功能——觉知层（S0）拒绝让识神编造。
- **直觉答案永远带"不可验真"标签。** S1（波）的会意只负责点火，旁边必附 S2（粒）的可证伪检查点——波粒并答，由你定夺。

## 5. 配置

- **火候/文化先验**：中立 / 儒 / 道 / 64卦 / 自定义——透明可审计，影响表达火候，不篡改事实。
- **感官器官**：在感官面板接入/断开任意基座（DeepSeek/Qwen/Llama/闭源API/视觉/听觉/具身/世界模型）——换基座，灵魂不动。
- **能耗账本**：设置中可见每次会话焦耳数——只认电表。

## 8. FAQ

**Q: 它比 GPT/Gemini 聪明吗？** A: 原始智力取决于接入的基座器官；它的差异不在更聪明，而在**可信**——会因果、有记忆、敢说不知道、答案可验真。
**Q: 离线能用吗？** A: 能。库与节点皆可全本地运行（CPU 瓦级血统），PWA 离线可用。
**Q: 我的数据去哪了？** A: 哪也不去。记忆只在你的设备/你的域里；导出 JSON 即完整带走。
**Q: 开发者如何让自己的模型「从太一汲取能力」？** A: 接认知层 API：你的模型作感官，太一回赠因果验证、记忆与凭证——你的模型即刻获得"诚实"。
