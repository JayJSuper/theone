### A. 定理级成立（可放心押注）
| 论纲主张 | 背书 |

### B. 成熟理论/强证据支持
- 双系统分工：双过程理论 + 神经符号混合（组合系统性缺陷的标准解）

## 三、论纲被低估的六个硬点（评审增补）

1. **路由器是隐藏的第一难题（homunculus 回归）。** "判断该快还是该慢"本身就是元认知——路由器若误把因果题送进 System 1，幻觉在上一层复活。修正：路由不该追求分类准确率，而要**成本敏感非对称**（因果题误入 S1 代价极高，统计题误入 S2 只是慢）→ 默认存疑、宁慢勿错；且路由器本身必须校准。
2. **"用 LLM 从文本抽因果图"有循环性。** S1 是统计鹦鹉，文本里民间因果与真因果混杂——抽取产物只是**假设图，不是知识图**。修正：每条边强制携带**证据等级**（已验证/文献支持/LLM 抽取/猜想），只有升级到足够等级的边才允许参与 do-查询。**这与我们项目的四档真实度纪律是同一个机制**——治理纪律即数据结构。
3. **可识别性检查应是一等公民。** 每个 do-查询前先跑 ID 判定：不可识别 → 拒答并指出缺哪个变量。这把"诚实"从伦理变成算法（见 A 节）。
4. **干预数据的经济学。** 自己做实验贵；但世界上**天然实验**（A/B 测试、政策变动、工具变量、断点）每天在发生。能从观测流中**识别并收割准实验**的引擎，比自己跑实验便宜一千倍——论纲应加这一收割层。
5. **变量从哪来：因果表征学习是最深的开放问题。** 论纲假设变量已给定（X→Y 三元组里 X、Y 是现成的词）。真正的科学前沿是从原始感知中学出**正确的变量**（Schölkopf 等"因果表征学习"，2021）。S1 的真正天赋恰在于此——它继承了人类亿万年命名好的变量库；但把这些变量**接地**到世界模型的状态空间，是 S1↔S2↔世界模型接口处的未解之谜。必须如实标注【假设·最深】。
6. **"一张世界因果图"是幻想，应改为模块化 SCM 库。** 背书：**独立因果机制原理**（ICM，Schölkopf/Peters/Janzing）——世界由可局部复用、可组合的小机制构成。架构形态应是"经验证的小图书馆 + 组合查询"，而非一张巨图。附带修正：现有 RL 世界模型（Dreamer 等）学的是 do(自己的动作)，**对动作是干预级的，对世界其他变量仍是预测级**——"世界模型=因果理解"要做此区分。
