一个从不假装知道的 AI。

02架构：双系统 + 觉知层

不是更大的 Transformer，而是 Transformer + 因果引擎的混合体——快慢分明、各司其职、由成本敏感路由调度，最上层是知道"自己知不知道"的元认知。

S0 元认知
校准与诚实层：七态不确定性、置信校准、拒答签发。一切输出经此盖印方可出门。拒答是可计算的（ID 判定）。弃权判定已实现
S2 因果引擎
显式因果图 + do-calculus + 反事实三步法 + 可识别性门。所有因果数字唯一来源——LLM 永不算因果。已实现·F-1 闭
S1 统计引擎
复用最强开源基座（DeepSeek/Llama/Qwen，热插拔）。负责理解、翻译、表达、创作——表达 S2 的数字，不改一个数。DeepSeek 已真实接入
S3 世界模型
物理模拟 + 多模态感知 + 连续预测。注：对自身动作是干预级，对世界变量是预测级——需与 S2 互补。M3+ 假设
横向支撑
证据分级因果图库（边带证据等级，够级才准 do-查询）· 主权记忆（查/改/删/带走）· 凭证生成/校验分离 · 成本敏感路由（宁慢勿错，疑似因果绝不落入 S1）v0 已实现

03核心算法

六个部件，三个有定理背书，全部可测试、可复算、可审计。

因果阶梯跃迁定理级
P(Y|X) → P(Y|do(X)) → P(Y_x | X=x′,Y=y′)
关联→干预→反事实。因果层级定理：仅凭关联数据原则上答不了干预问题——这是 The One 与一切纯 LLM 的数学分界线。
可识别性门定理级
ID(query, graph) ∈ {identifiable, NOT} → answer | abstain
do-calculus 完备性 + ID 算法：查询不可识别时，系统能在数学上证明自己答不了——"因果的不知道"是可计算的。诚实不靠自觉，靠判定算法。
校准弃权定理级
answer if conf ≥ τ(calibrated) else ABSTAIN
Chow 拒识规则：弃权可证地换错误率。幻觉有数学下界（Kalai-Vempala），但可以转换为可控的沉默。血统：99.6% 弃权精度。
认知凭证已实现
receipt = {method, graph_hash, adj_set, numbers, tier, limits}
验证比生成廉价（Proof-Carrying 范式）：信任成本从"重跑模型"降为"查验收据"。生成器与校验器零共享代码、分离部署。
成本敏感路由v0 已实现
misroute(causal→S1) ≫ misroute(stat→S2) ⇒ 宁慢勿错
路由不追求分类准确率，追求非对称损失最小：因果题误入 S1 会让幻觉复活；存疑且无验证图谱时直接弃权。
证据分级图库v0 已实现
tier ∈ {conjecture, llm_extracted, literature, machine_validated}
LLM 抽出的因果边只是假设，不是知识——只有升到足够证据级的边才许参与 do-查询。治理纪律即数据结构。

05应用场景

凡是"答错代价高"的地方，就是 The One 的主场——因为它敢不答。

医疗辅助

疗效因果评估、混杂识别、证据不足时明确拒答——拒答在医疗场景是救命特性。

金融与审计

风险因果链、反事实压力测试（"如果跌 40%"）、全量凭证可出庭级追溯。

政策评估

P(结果|do(政策))——干预问题正是政府每天在问而 LLM 原则上答不了的问题。

科学研究

假设生成→实验设计→预注册→负结果照发。仿真农场即服务。

企业决策

"如果当时没降价销量会怎样"——反事实复盘 + 下一步 A/B 实验设计。

个人终身 AI

记住你十年、随你成长、记忆永远可带走——而且从不对你编造。

08理论根基

"在干预与反事实问题、以及可信度维度上，它不是可能更好——是原则上更好；在纯生成维度上，它不试图更好，直接借用最好的。"

因果层级定理

Pearl/Bareinboim：关联数据原则上答不了干预问题。存在一类问题，纯 LLM 永远答不了、The One 答得了——这就是存在性证明。

校准-幻觉下界 × Chow 弃权

Kalai-Vempala：幻觉对 LLM 不可根除。Chow 规则：弃权可证地换错误率。不可避免的错误 → 可控的沉默。

验证-生成不对称

查验一张收据远廉价于重跑一次生成——这是凭证协议能成为"认知界 HTTPS"的复杂度根基。

诚实清单（理论不保证的）：自动因果发现欠定（所以主动实验是必需品）；S1 质量无定理；"净增益>脚手架"是经验赌注（Baseline B 待测）；变量接地（因果表征学习）是最深的开放问题。三大科学赌注：①干预鸿沟（首测 31×，限定域）②反事实梯度泛化（T4 强判据 FAIL，弱判据过）③主动层安全配重（设计冻结）。
